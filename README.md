## О проекте

**TaskFlow** решает проблему разрозненных заметок и потерянных задач у фрилансеров и небольших команд. Приложение предоставляет наглядную Kanban-доску прямо в браузере.

**Ключевые особенности:**
- 🔒 Полная приватность — все данные хранятся локально в браузере (`localStorage`)
- ✈️ Работает без интернета — не нужен сервер или облако
- ⚡ Мгновенный старт — достаточно открыть `index.html`
- 🎯 Ноль зависимостей — никаких `npm install`

## Функциональность

| Функция | Описание |
|---|---|
| ➕ Создание задач | Название, описание, теги, приоритет, статус, дедлайн |
| 📋 Kanban-доска | Три колонки: «К выполнению», «В процессе», «Выполнено» |
| ✏️ Редактирование | Изменение любого поля, смена статуса |
| 🗑️ Удаление | Удаление отдельной задачи или всех выполненных |
| 🔍 Поиск | По названию, описанию, тегам в реальном времени |
| 🔽 Фильтрация | По приоритету (Высокий / Средний / Низкий) |
| ↕️ Сортировка | По дате создания, приоритету или дедлайну |
| ⚠️ Просрочка | Визуальное выделение задач с истёкшим дедлайном |
| 📊 Прогресс | Прогресс-бар и счётчики задач в шапке |
| 💾 Сохранение | Автоматическое сохранение при каждом изменении |

## Быстрый старт

### Вариант 1 — открыть напрямую (самый простой)

1. Скачайте или клонируйте репозиторий:
   ```bash
   git clone https://github.com/mitya080901/TaskFlow.git
   cd taskflow
   ```
2. Откройте файл `index.html` в браузере двойным кликом.  
   Готово — приложение работает.

### Вариант 2 — через локальный сервер (рекомендуется)

Если вы работаете с файлами через терминал, можно поднять простой HTTP-сервер:

```bash
# Python 3
python3 -m http.server 8080

# Node.js (npx)
npx serve .
```

Затем откройте в браузере: `http://localhost:8080`

### Требования

- Современный браузер: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- Интернет-соединение нужно **только** для загрузки шрифтов Google Fonts при первом открытии.  
  Без интернета приложение работает полностью — используются системные шрифты.

## Архитектура

Приложение построено по принципу **модульного vanilla JS** без внешних зависимостей.

```
app.js
├── Константы & модель данных (Task)
├── Хранилище (loadTasks, saveTasks → localStorage)
├── CRUD (createTask, updateTask, deleteTask, clearDoneTasks)
├── Фильтрация & сортировка (getFilteredTasks)
├── Рендеринг (render, renderTaskCard)
├── UI: форма (openFormModal, handleSave)
├── UI: просмотр (openViewModal)
├── Уведомления (showToast)
└── Инициализация & обработчики событий (init)
```

**Поток данных:**
```
Действие пользователя → Обработчик события → CRUD-функция → saveTasks() → render()
```

Все данные хранятся в `state.tasks` (in-memory массив) и синхронизируются с `localStorage` при каждом изменении.
